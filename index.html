BetaGet museum hours

Get upcoming museum operating hours

Authorizations:
MuseumPlaceholderAuth
query Parameters
startDate
string <date>
Example: startDate=2023-02-23

The starting date to retrieve future operating hours from. Defaults to today's date.

page
integer
Default: 1
Example: page=2

The page number to retrieve.

limit
integer <= 30
Default: 10
Example: limit=15

The number of days per page.

Responses

https://api.freshcraft.org/v1/museum-hours

page
integer

limit

Buy tickets

Museum tickets for general entrance or special events.

Buy museum tickets

Purchase museum tickets for general entry or special events.

Authorizations:
MuseumPlaceholderAuth
Request Body schema: application/json
required
ticketType
required
string (TicketType)
Enum: "event" "general"

Type of ticket being purchased. Use general for regular museum entry and event for tickets to special events.

eventId
string <uuid> (EventId)

Unique identifier for a special event. Required if purchasing tickets for the museum's special events.

ticketDate
required
string <date> (Date)

Date that the ticket is valid for.

email
required
string <email> (Email)

Email address for ticket purchaser.

phone
string (Phone)

Phone number for the ticket purchaser (optional).

Responses

https://api.freshcraft.org/v1/tickets

Request samples

  • Payload
Content type
application/json
Example
{
  • "ticketType": "general",
  • "ticketDate": "2023-09-07",
  • "email": "todd@example.com"
}

Response samples
